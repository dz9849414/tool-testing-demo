## 📦 使用步骤

### 1. 执行数据库脚本
```sql
-- 在MySQL中执行
source database-schema.sql
```

### 2. 配置MyBatis Plus
确保 `application.yml` 中配置了Mapper扫描：
```yaml
mybatis-plus:
  mapper-locations: classpath*:/mapper/**/*.xml
  type-aliases-package: com.example.tooltestingdemo.entity
```

### 3. 配置Mapper扫描
确保主类或配置类上有 `@MapperScan`：
```java
@MapperScan("com.example.tooltestingdemo.mapper")
```

### 4. 使用示例
```java
// 注入Service
@Autowired
private InterfaceTemplateService templateService;

// 创建模板
InterfaceTemplate template = new InterfaceTemplate();
template.setName("测试接口");
template.setProtocolType("HTTP");
template.setMethod("POST");
template.setBaseUrl("http://localhost:8080");
template.setPath("/api/test");

templateService.createTemplate(template, headers, parameters, 
    formDataList, assertions, preProcessors, postProcessors, variables);
```
